# Adith's Personal Portfolio

Welcome to the repository for my personal portfolio website! This project serves as a showcase of my skills, projects, and services as a full-stack web developer and video editor. 

## 🌟 Features

*   **Modern UI/UX Design**: A sleek, dark-themed interface featuring glassmorphism elements and vibrant cyan/purple neon accents.
*   **Dynamic Animations**: 
    *   **Typed.js** integration for an engaging typing animation in the Hero section.
    *   Smooth **Scroll Reveal** animations that bring elements to life as you navigate down the page.
*   **Fully Responsive**: Carefully designed to look great on all devices, from large desktop monitors to mobile phones.
*   **Clean Structure**: Built from scratch without heavy frameworks for maximum performance and easy customization.

## 🛠️ Tech Stack

*   **HTML5**: Semantic markup and layout structure.
*   **CSS3**: Advanced styling including Flexbox, CSS Grid, custom variables, and keyframe animations.
*   **JavaScript (Vanilla)**: DOM manipulation, event listeners for mobile navigation, and scroll-triggered animations.

## 🚀 How to Run Locally

Since this is a static website, you don't need any complex backend servers to run it.

1.  Clone this repository to your local machine:
    ```bash
    git clone https://github.com/adithx2/Adith-personalportfolio.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd Adith-personalportfolio
    ```
3.  Simply double-click the `index.html` file to open it in your default web browser, or use an extension like VS Code Live Server for hot reloading.
